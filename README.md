## installation
```shell
npm install -S @jswork/react-rte-ui
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-rte-ui/dist/style.css";

  // or use sass
  @import "~@jswork/react-rte-ui/dist/style.scss";
  @import "~@jswork/wsui-rte-icons";
  @import "~balloon-css";

  // customize your styles:
  $react-rte-ui-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
